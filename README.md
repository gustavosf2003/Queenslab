## Development

> Use o [yarn](https://yarnpkg.com/) as default package manager.


## Project setup
```
yarn install
```

### Start the project
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Run your tests
```
yarn test:e2e
```

### Lints and fixes files
```
yarn lint
```
